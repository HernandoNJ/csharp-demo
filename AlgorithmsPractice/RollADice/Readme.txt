Project requirements:
- The application starts by generating a random number and showing an initial text
    - Generate a number from 1 t0 6 that must be saved internally and hidden from the user
    - Initial text requesting the user to enter a number
- The user has 3 tries to guess the number
    - If the user guesses the number, a "You win" text is displayed
    - If the user doesn't guess the number, a "Try again" text is displayed
- The application finishes in 2 cases
    - The user guesses the number
    - The user couldn't guess the number after 3 tries
